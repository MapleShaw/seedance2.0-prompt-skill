# Seedance 2.0 视频提示词生成器

一个为字节跳动**即梦 Seedance 2.0** 平台量身打造的 Cursor Agent Skill，帮你生成可直接使用的专业中文视频提示词。

---

## 这是什么

即梦 Seedance 2.0 是字节跳动推出的多模态 AI 视频生成模型，支持图片、视频、音频多模态混合输入，能生成 4-15 秒的高质量视频。

好的提示词和普通提示词，生成效果天差地别。这个 Skill 封装了平台的完整能力体系和最佳实践，让你告别"瞎写提示词"，直接产出高品质方案。

本 Skill 现在包含 **五大能力模块**：

| 模块 | 解决什么 |
|:---|:---|
| 📷 **相机四维编码系统**（Z/Y/X/F） | 从"关键词记忆"升级为"坐标思维"，精准控制任何镜头 |
| 🎨 **深度美学约束** | 对标 Octane/V-Ray 渲染品质 + 6 套冷暖色调系统 + 极简先锋构图 |
| 🎬 **长视频生产流水线** | 角色卡 → 分镜 → 逐镜头的完整前期制作流程（含 25 格流水线） |
| ✂️ **AI 素材剪辑节奏** | 六套剪辑公式 + AI 素材首尾修剪/色彩统一/畸变遮掩对策 |
| 🖼️ **图片生成** | 角色卡片图、首帧图、关键帧图的专业提示词规范 |

## 提示词结构速查

### 基础结构（≤12秒）
```
[风格/色调总纲]，[主体描述]，[动作序列]，[环境/光影]，[镜头语言]，[音效描述]
```

### 时间戳分镜（13-15秒，强烈推荐）
```
[时长][风格总纲]，
0-3秒：[画面 + 镜头 + 音效]；
4-8秒：[画面 + 镜头 + 音效]；
...
```

### 史诗/大制作结构
```
[时长][品质锚定]，[核心氛围宣言]，[大气连贯声明]，
0-Xs：[画面] + [运镜] + [可选帧率]；
...
光影：[光源层] + [光行为层] + [色调层]，
[收束句] + [张力宣言]。
```

## 平台关键限制

- **单次生成上限 15 秒**，超出需分段拼接
- **混合文件上限 12 个**（图片 + 视频 + 音频合计）
- **@引用必须用官方命名**：`@图片1`~`@图片9`、`@视频1`~`@视频3`、`@音频1`~`@音频3`
- **禁止写实真人面部素材**，平台会自动拦截
- 图片：jpeg/png/webp 等，≤9张，单张 <30MB
- 视频：mp4/mov，≤3个，总时长 2-15秒
- 音频：mp3/wav，≤3个，总时长 ≤15秒

## 超长视频最佳实践（>15秒）

Seedance 2.0 生成长视频时，关键不是把提示词写得更长，而是做好**结构、锁定、节奏、衔接**四件事。

**推荐工作流：**

1. **先定三段式结构**：建立世界 → 变化推进 → 情绪收束
2. **每段都加一致性锁定**：
   - 全片保持统一视觉风格，风格不漂移
   - 同一角色保持一致五官、身份、服装和发型
   - 同一场景保持一致环境、光线、背景与空间关系
3. **延长段开头先做桥接**：优先保留 0.8-1.5 秒风、光影、尘埃、呼吸等微动
4. **连续剧情优先用尾帧延续法**：接上一段最后一帧的构图与状态，前 1 秒只做微动，再进入新动作
5. **统一负面提示词**：防止风格漂移、角色变脸、偏色、光线突变、无关人物和文字水印

**推荐负面限制：**
```
禁止风格漂移，禁止角色变脸或换人，禁止突然偏色，禁止新增无关人物，
禁止光线突变，禁止出现文字、字幕、LOGO、水印
```

更完整的分段模板与示例见 `references/long-video-strategy.md`。

## 使用方式

在 Cursor 中直接用自然语言描述需求即可，例如：

- "帮我写一段 15 秒的仙侠战斗视频提示词"
- "我有一张产品图，想做一个旋转展示的广告，竖屏 9:16"
- "参考这个视频的运镜风格，生成一段城市延时摄影"
- "帮我把这段 8 秒视频延长到 15 秒"
- "帮我做一个 30 秒的剧情短片，按分段续拍，保持同一人物和场景不漂移"
- "我想要一种压迫感的镜头，主角正面近景" → 自动查情绪速查表返回坐标公式
- "帮我规划一条 60 秒的赛博朋克产品广告，包含剪辑节奏方案"
- "这条视频我已经生成了 6 条素材，该怎么剪？" → 按六套公式推荐方案

Skill 会引导你确认时长、比例、素材情况，然后生成 2-3 个不同风格版本供选择，每个版本都可以直接复制到即梦平台使用。

## 进阶技巧

**大制作质感的四个核心：**

1. **品质锚定开头**：用 `UnrealEngine5渲染，工业光魔级VFX特效` 比泛词"电影感"效果更精准
2. **大气连贯声明**：在总纲后声明全片统一物理效果（如"每帧都有薄雾弥散效果"），防止分镜间氛围断裂
3. **光影三层结构**：光源 → 光行为 → 色调，三层各司其职
4. **镜头污染增强真实感**：雾粒粘镜、雨水附着镜头前等"不完美"效果反而提升沉浸感

**运镜坐标编码（四维系统）：**
- 核心思维：不再记忆零散关键词，而是建立坐标系 → 先定位 Z+Y+X → 再叠加 F → 最后翻译成自然语言
- **Z 距离**：Z1 大特写 → Z9 大远景（决定看清毛孔还是看清世界）
- **Y 高度**：Y1 虫视 → Y7 顶视（决定仰视崇拜还是俯视压制）
- **X 方位**：X1 正面 → X4 背面（决定立体感与心理距离）
- **F 滤镜**：焦段 + 景深 + 畸变 + POV/OTS + 构图几何
- **铁律**：每个镜头最多双轴运动；Z1-Z3 + X 轴大幅旋转 = 崩脸陷阱
- 14 种情绪 → 坐标速查表直接给出公式，详见 [camera-codec.md](references/camera-codec.md)

**剪辑节奏公式（>15 秒必读）：**
- 六套公式速选：呼吸式（日常）/ 心跳式（紧张）/ 海浪式（情感）/ 子弹时间（高潮）/ 脉冲式（快闪）/ 静默锤击（先锋）
- AI 素材五大对策：首尾修剪 / 色彩统一 / 转场弥合 / 快节奏遮掩畸变 / 生成 2-3 倍素材库
- 详见 [editing-rhythm.md](references/editing-rhythm.md)

**长视频补充原则：**
- 节奏优先遵守「慢 → 稍快 → 慢」
- 分段之间优先保持同构图推进，而不是频繁换场景
- 声音连续性很重要，优先用环境音或同一首音乐分层做桥接
