# 长视频生产流水线（>15秒）

> 长视频不是"长的短视频"。≤15秒可以靠一条提示词搞定，>15秒必须走完整的前期制作流程。跳过任何一步都会导致成片质量断崖式下降。

---

## 完整流程

```
① 概念/剧本 → ② 角色设计 → ③ 角色卡片图 → ④ 分镜脚本 → ⑤ 分镜参考图 → ⑥ 逐镜头视频生成 → ⑦ 后期拼接
```

### ① 概念/剧本

- 一句话说清楚"这个视频讲什么"
- 明确总时长、分几段、每段的叙事任务
- 使用三段式骨架：建立世界 → 变化推进 → 情绪收束

### ② 角色设计

- 确定角色数量、各角色的外观特征（五官、发型、服装、体型、标志性道具）
- 写出角色描述文档，供后续所有环节引用
- 关键：描述要足够具体，以确保在多次生成中角色外观一致

### ③ 角色卡片图生成

- 用即梦的**图片生成**功能，为每个角色生成多角度参考图
- 推荐生成：正面、侧面、全身、半身各一张
- 这些图片将作为 @图片N 在后续视频生成中引用，是角色一致性的锚点
- **角色卡片图的质量直接决定成片中角色的一致性**

### ④ 分镜脚本

- 按时间轴拆分为具体镜头（每个镜头 ≤15秒）
- 每个镜头写明：画面内容、运镜方式、角色动作、情绪、音效/台词
- 标注镜头之间的衔接方式（桥接镜头/同构图推进/时间流逝）
- 分镜脚本 = 视频的蓝图，写得越细，后续越可控

### ⑤ 分镜参考图

- 为关键镜头生成参考图片（首帧/关键帧）
- 用即梦图片生成，提示词基于分镜脚本描述
- 参考图的作用：
  - 首帧图：锚定每段视频的起始画面
  - 关键帧图：引导模型理解特定画面构图
  - 风格参考图：统一全片视觉调性

### ⑥ 逐镜头视频生成

- 第1段：首帧图 @图片N + 角色参考 @图片N + 提示词 → 正常生成
- 第2段起：上一段视频 @视频1 + 角色参考 @图片N + 延长提示词 → 视频延长
- 每段都重复锁定语句（风格/角色/场景）
- 每段都附加负面提示词
- **不满意就重新生成该段，不要将就**——一段凑合会拖垮全片

### ⑦ 后期拼接

- 使用剪映/PR/达芬奇等工具拼接各段
- 添加转场（如果段间不够丝滑）
- 统一调色确保各段色调一致
- 添加音乐/音效/字幕
- 最终校验节奏和连贯性

## 各步骤的提示词模板

### 角色卡片图提示词模板

```
[风格声明]，[角色名/描述]的角色设定图，
正面半身像，五官清晰，[具体外貌描述：发型、发色、瞳色、肤色]，
穿着[具体服装描述]，[具体配饰/道具]，
表情[默认表情]，[背景：纯色/简洁]，
角色设计参考图风格，高清细节，多角度一致性
```

### 分镜参考图提示词模板

```
[风格声明]，[该镜头的画面描述]，
[角色]@图片N在画面中的位置和姿态，
[运镜暗示的构图：如仰拍则低角度构图]，
[光影和氛围]，[前景/背景层次]，
电影分镜概念图风格
```

### 逐镜头视频提示词

第1段沿用 SKILL.md 中的时间戳分镜法或史诗大制作结构。
第2段起沿用 long-video-strategy.md 中的视频延长模板。

## 常见问题

**Q：角色卡片图生成出来不够一致怎么办？**
A：多生成几张，挑最好的一张作为锚点。后续所有视频生成都引用同一张角色图。一致性来自"反复引用同一张参考图"，而非"希望模型自己记住角色"。

**Q：分镜参考图和首帧图有什么区别？**
A：首帧图是"视频从这张图开始动"，需要画面精确；分镜参考图是"给模型看一下这个镜头大概长什么样"，可以更概念化。

**Q：>60秒的视频怎么做？**
A：拆分为独立场景分别生成（每个场景 ≤30-45秒），最后用剪辑软件拼接。不要试图用视频延长做超过60秒的连续视频——漂移风险太高。

**Q：可以跳过某些步骤吗？**
A：角色卡片图和分镜参考图是最不能跳过的。跳过它们 = 把一致性完全交给运气。
